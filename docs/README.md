# Clubhuis App

Private PWA for a friend group.

## Stack

- Next.js
- TypeScript
- Firebase Auth
- Firestore
- Firebase Storage
- TailwindCSS

## Features

- Google Login
- Photo Sharing
- Events
- Drinking Games
- Statistics
- Achievements

## Development Rules

- Use TypeScript
- Use Firebase SDK v10+
- Use Firestore Security Rules
- Mobile First Design
- PWA Ready
