# User Stories

## Authenticatie

Als gebruiker wil ik kunnen aanmelden via Google zodat ik geen wachtwoord hoef te onthouden.

## Feed

Als gebruiker wil ik foto's kunnen delen zodat herinneringen bewaard blijven.

Als gebruiker wil ik kunnen reageren op foto's zodat ik met vrienden kan communiceren.

## Events

Als gebruiker wil ik evenementen kunnen plannen zodat iedereen weet wanneer we samenkomen.

## Statistieken

Als gebruiker wil ik mijn drankjes kunnen registreren zodat statistieken bijgehouden worden.

## Games

Als gebruiker wil ik drinking games kunnen starten zodat we ons kunnen amuseren tijdens bijeenkomsten.

## Profiel

Als gebruiker wil ik mijn persoonlijke statistieken bekijken zodat ik mijn records kan zien.
