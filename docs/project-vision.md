# Clubhuis App

## Doel

Een privé Progressive Web App voor onze vriendengroep.

Gebruikers kunnen:
- Inloggen via Google
- Foto's uploaden
- Evenementen beheren
- Drinking games spelen
- Statistieken bekijken
- Herinneringen bewaren

## Technologie

Frontend:
- Next.js
- TypeScript
- TailwindCSS

Backend:
- Firebase Authentication
- Firestore
- Firebase Storage

Hosting:
- Firebase Hosting

## Doelplatformen

- Android
- iPhone
- Desktop


# Project Vision

## Missie

Een privé digitaal platform creëren voor onze vriendengroep.

Het platform moet:

- Leuke herinneringen bewaren
- Activiteiten organiseren
- Statistieken verzamelen
- Sociale interactie stimuleren
- Entertainment aanbieden via games

## Doelgroep

De leden van de Clubhuis vriendengroep.

## Belangrijkste principes

### Simpel

Iedereen moet de app onmiddellijk kunnen gebruiken.

### Mobiel

De app wordt voornamelijk gebruikt op smartphone.

### Fun

Entertainment staat centraal.

### Privé

Alle gegevens zijn enkel zichtbaar voor leden van de groep.

## Succescriteria

- Minimaal 10 actieve gebruikers
- Wekelijks gebruik
- Foto's en herinneringen worden actief gedeeld
- Games worden gebruikt tijdens bijeenkomsten
