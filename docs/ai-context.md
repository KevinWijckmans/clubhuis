# AI Development Context

Dit project is een Progressive Web App voor een vriendengroep genaamd Clubhuis.

## Doel

Een privé sociaal platform bouwen met:

- Google Authentication
- Foto Sharing
- Events
- Drinking Games
- Statistieken
- Achievements

## Technologie

- Next.js
- TypeScript
- Tailwind CSS
- Firebase

## Code Standaarden

- Gebruik TypeScript
- Gebruik function components
- Gebruik Firebase SDK v10+
- Gebruik async/await
- Vermijd any types
- Gebruik server actions waar mogelijk

## Database

Firestore wordt gebruikt als primaire database.

## Design

Mobile First
Dark Mode
Modern Social App uiterlijk

## Doelgebruikers

Vriendengroep van ongeveer 10 tot 30 personen.
